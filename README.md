Walk Alone Game
===============

Requerimientos:  [node](http://nodejs.org/) y [bower](http://bower.io/).

Instrucciones
-------------

 1. `~ git clone https://github.com/GrupoW/base-cli.git`
 2. `~ cd walk-alone`
 3. `~ npm && bower install`
 4. `~ gulp`
 
 Controles
 ---------
  - Presionar `j` para saltar
  - Presionar `f` para disparar
